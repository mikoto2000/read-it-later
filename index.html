プライバシーポリシー

本サービスでは、以下の方針に基づき、個人情報の取り扱いを行います。

1. 取得する情報

  • GitHub アカウントのユーザー名（公開プロフィール名）
  • GitHub 認証時に Supabase が内部的に保持する認証情報（メールアドレス等）

2. 利用目的

  • ユーザー名は、画面上に表示するためにのみ利用します
  • 取得した情報は Supabase に保存していません

3. 外部サービスとの連携（GitHub）

  • 当サービスは GitHub アカウントによるログイン機能を提供しています
  • GitHub のプライバシーポリシーは こちら をご確認ください

4. 第三者提供について

  • 取得した情報を第三者に提供することはありません

5. クッキー等の利用

  • 本サービスはログインセッション管理のために、ブラウザに Cookie やローカルストレージを利用することがあります

6. プライバシーポリシーの変更について
